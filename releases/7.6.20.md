## Arches 7.6.20 Release Notes

### Bug Fixes and Enhancements

-  Handles cases where the edittype key is not defined in the edit_lookup_table  #[12536](https://github.com/archesproject/arches/pull/12536)
-  [SECURITY] Fixes issue with tile permissions


### Dependency changes:

```
Python:
    Upgraded:
        None
JavaScript:
    Upgraded:
        None
```

1. Upgrade to version 7.6.0 before proceeding by following the upgrade process in the [Version 7.6.0 release notes](https://github.com/archesproject/arches/blob/dev/7.6.x/releases/7.6.0.md)

1. Upgrade to Arches 7.6.20

    ```
    pip install --upgrade arches==7.6.20
    ```

1. If you are running Arches on Apache, restart your server:
    ```
    sudo service apache2 reload
    ```
